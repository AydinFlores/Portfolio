# Portfolio
Frontend portfolio focused on modern, clear, and conversion-ready landing pages. Built with HTML, CSS, and JavaScript, with ES/EN support and lightweight interactions.

## Highlights
- Responsive UI with custom theming (light/dark).
- ES/EN language toggle with JSON i18n.
- Projects showcase with image previews and hover video.
- Clean, focused contact CTA pointing to Fiverr.

## Structure
- `index.html` - main layout and sections
- `style.css` - styling and responsive rules
- `main.js` - interactions, i18n loader, and UI logic
- `i18n/` - translation files (`es.json`, `en.json`)
- `assets/` - images and videos
